Troubleshooting
Due to the limitations of Chrome 33+,
1. This extension only works when Google is the default search engine.
2. Try to disable other new tab page related apps if this one does not load.
3. Try to go to settings -> Show advanced settings... -> Enable "Use a prediction service to help complete searches and URLs typed in the address bar".
This is a limitation in the chrome browser that extensions can not open apps in the same tab.
